# Natural Instructions Dataset (v2.8)

This directory is intended to hold the Natural Instructions (Super-NaturalInstructions) dataset.

## How to download the full dataset

You can download the full dataset from the official repository:
[https://github.com/allenai/natural-instructions](https://github.com/allenai/natural-instructions)

To get the latest version and all tasks, you can clone the repository:

```bash
git clone https://github.com/allenai/natural-instructions.git
```
